About

NetBeans is an integrated development environment (IDE), often used for Java. It's easy to use and popular, with easy-to-learn shortcuts and commands. That makes it a great IDE for Java development! And it's not limited to Java, either. You can download versions with support for other languages too.

Keboard Shortcuts in NetBeans

While there are many keyboard shortcuts in NetBeans, there are a few you will find incredibly useful for Java development.

Shortcut Result
Alt-Shift-F Format Code
F9 Compile package/file
Ctrl+/ Comment/un-comment
Ctrl+E Delete current line
