About NetBeans

NetBeans is the official integrated development environment (IDE) for Java 8. It's popular, with easy-to-learn shortcuts and commands, making it a great IDE for Java development!

NetBeans includes a GUI builder for Java SE, making it easy to build graphical interfaces.

It's not limited to Java, either. You can download versions with support for other languages like HTML, JavaScript, and PHP. You can even use it for C/C++ development!

Why use an IDE?

Integrated development environments (IDEs) combine text editors with more advanced features, like syntax code highlighting, and the ability to rename files without breaking things (refactoring). IDEs will often also have shortcuts and keyboard combinations to make your life easier (like a command to comment out code, for example). IDEs have debugging tools as well, including in-line checking that highlights bad code, and let you run your code step-by-step.

Beyond that, IDEs like NetBeans have a visual file tree, allowing you to see how files are related and what directory they are in. This makes it easy to see if all the needed files are in the right place when it comes time to compile.

Strengths

NetBeans is intuitive and easy to use, even for beginners. One can install and use it right out of the box. It has an easy to use Swing GUI design tool allowing drag and drop. It has auto completed code and gives programmers a list of possible code fragments to choose from. It debugs and gives hints on improving code.


Weaknesses

It needs more memory than lighter IDEs.The advanced tools and features require some training.

History

NetBeans began in 1996 as a student project in the Czech Republic. The project's original name was Xelfi. After some success a few years later they decided to make it a commercial venture. NetBeans became the first Java IDE written in Java. In 1999 Sun Microsystems expressed interest and NetBeans became the "flagship" tool for the creators of Java.

Plugins in NetBeans

Plugins are used to extend NetBeans' functionality and interface. They may add new color themes, tools, or support for more types of files.

Popular plugins for NetBeans
  • extension Darcula LAF Darcula LAF is a Look-and-Feel theme plugin coming from the IntelliJ IDEA theme
  • extension JS CSS Minify Compress This plugin minifies and compresses JavaScript, CSS and HTML to decrease page load times.
  • extension JSLint JSLint is a well-know code-quality tool and website, used to find bugs and typos in JavaScript.
  • extension ez-on-da-ice This plugin allows you to create an easy for the eyes look and feel of the NetBeans IDE by changing the colors. >
  • extension Additional Java Hints As the name suggests Additional Java Hints provides additional hints that allows you to write better quality code and points out any potential issues.
  • extension EasyPMD EasyPMD is a code analyzer that scans Java source code looking for over 400 potential problems.
  • extension easyUML “easyUML is a UML diagram tool currently supporting only class diagrams providing drag-and-drop visual editor, Java code generation from diagrams and ‘reverse engineering’ to create class diagrams from Java code. It is very suitable for teaching software design.”
